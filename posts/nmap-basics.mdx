Nmap remains a foundational capability for network recon.

```bash
nmap -sC -sV -oA baseline 10.10.10.0/24
```

Key flags:

- `-sC`: default scripts
- `-sV`: version detection
- `-oA`: output all formats with prefix

Use thoughtfully; always ensure authorization.
